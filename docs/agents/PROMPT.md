  👉 [docs/guidelines/CACHE_GUIDELINE.md](../guidelines/CACHE_GUIDELINE.md)

- **Mọi thắc mắc hoặc đề xuất bổ sung, vui lòng liên hệ team kiến trúc hoặc cập nhật trực tiếp vào guideline.**
